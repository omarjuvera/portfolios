How to run Kali Linux in Windows

  1. Open the Microsoft Store app to install Windows Subsystem for Linux (WSL)


    This what allows Windows install Linux distributions
     
  2. Once WSL is installed (you may have to reboot), open the Microsoft Store app once more to install Kali Linux
     
  3. As you can see in the screenshot if you look up "Linux" in the Microsoft Store app you will find various Linux distributions besides Kali, like Ubuntu.
     
  4. ENJOY!

Bonus app!
While you're in Microsoft Store, I highly recommend you to download the Microsoft app Windows Terminal. This is a great replacement for the old Command Prompt
Windows Terminal allows you to have multiple command-line applications like Kali Linux, PowerShell, Command Prompt, etc. in tabs instead of multiple windows. Similar to a web browser.
